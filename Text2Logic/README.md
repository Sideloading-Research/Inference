## üìã Overview

**Text2Logic** is a comprehensive system for converting natural language text to formal logic notation and performing exhaustive logical inference. It combines natural language processing with a complete logical inference engine to automatically extract knowledge and derive all possible conclusions.

### Key Features

- ‚ú® **Natural Language Processing**: Convert Spanish text to formal logic
- üîç **Logical Operator Detection**: Recognizes "y", "o", "si...entonces", "si y solo si"
- üß† **Exhaustive Inference**: Derives all possible conclusions using 9+ inference rules
- üîó **Forward Chaining**: Data-driven inference with saturation
- üìä **Explainability**: Tracks reasoning chains for every derived fact
- üöÄ **Three Usage Modes**: Direct script, internal API, or library import
- ‚úÖ **Automatic Verification**: Checks Ollama and Gemma installation

```python

# Convert text to logic
```

## üß† Inference Rules

Text2Logic implements 9 fundamental inference rules:

### 1. Modus Ponens
```
A ‚Üí B, A ‚ä¢ B
```
If A implies B, and A is true, then B is true.

### 2. Modus Tollens
```
A ‚Üí B, ¬¨B ‚ä¢ ¬¨A
```
If A implies B, and B is false, then A is false.

### 3. Hypothetical Syllogism
```
A ‚Üí B, B ‚Üí C ‚ä¢ A ‚Üí C
```
Chain implications together.

### 4. Disjunctive Syllogism
```
A ‚à® B, ¬¨A ‚ä¢ B
```
If A or B, and not A, then B.

### 5. Simplification
```
A ‚àß B ‚ä¢ A, B
```
From conjunction, extract individual facts.

### 6. Conjunction
```
A, B ‚ä¢ A ‚àß B
```
Combine individual facts.

### 7. Resolution
```
¬¨A ‚à® B, A ‚à® C ‚ä¢ B ‚à® C
```
Fundamental rule for theorem proving.

### 8. Biconditional Elimination
```
A ‚Üî B ‚ä¢ (A ‚Üí B) ‚àß (B ‚Üí A)
```
Convert biconditional to implications.

### 9. Variable Unification
```
(X)IsA(estudiante), (Pedro)IsA(estudiante) ‚ä¢ X=Pedro
```
Match variables with constants.

## üêõ Troubleshooting

### "Cannot connect to Ollama"

**Solution:**
```bash
# Start Ollama service
ollama serve
```

### "Model gemma:2b not found"

**Solution:**
```bash
# Pull the model
ollama pull gemma:2b
```

### "Conversion produces no results"

**Possible causes:**
- Ollama not responding ‚Üí Check `ollama serve` is running
- Model timeout ‚Üí Increase timeout in `text_to_logic.py`
- Prompt not understood ‚Üí Try rephrasing input text

### "Too many facts derived"

**Solution:**
```python
# Disable conjunction rule to reduce combinatorial explosion
engine = LogicEngine(enable_conjunction=False)
```
